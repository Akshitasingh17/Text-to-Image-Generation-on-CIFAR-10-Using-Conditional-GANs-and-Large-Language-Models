# Text-to-Image-Generation-on-CIFAR-10-Using-Conditional-GANs-and-Large-Language-Models
This project aims to explore text-to-image generation by leveraging Conditional Generative Adversarial  Networks (cGANs) in conjunction with Large Language Models (LLMs) to generate realistic images  conditioned on textual input derived from the corresponding CIFAR-10 class labels or more descriptive  prompts.
